# 第2章 信息的表示和处理

## 核心概念

### 信息存储
- **无符号整数**: 非负整数
- **有符号整数**: 使用补码表示
- **浮点数**: IEEE 754 标准

### 数据表示
- **十六进制表示**: 便于表示位模式
- **字长**: 决定了指针大小和最大地址
- **字节序**: 大端序和小端序

### 整数运算
- **无符号运算**: 模 2^w 运算
- **有符号运算**: 补码运算
- **溢出**: 当结果超出表示范围时发生

## 难点解析

### 补码表示
- 正数: 最高位为0
- 负数: 最高位为1
- 范围: -2^(w-1) 到 2^(w-1)-1

### 浮点数表示
- 符号位(s): 1位
- 阶码(e): 8位(单精度)或11位(双精度)
- 尾数(m): 23位(单精度)或52位(双精度)

### IEEE 754 标准
- 规格化值
- 非规格化值
- 特殊值(无穷大、NaN)

## 代码示例

```c
// 位操作示例
int get_msb(int x) {
    // 获取最高有效位
    return (x >> 31) & 1;
}

// 判断是否为2的幂
int is_power_of_2(int x) {
    return x > 0 && (x & (x - 1)) == 0;
}
```
